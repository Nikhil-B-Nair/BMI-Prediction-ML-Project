```
# BMI Prediction Using Machine Learning
```

## `## Overview` 

```
This project predicts Body Mass Index (BMI) using a Linear Regression model
trained on Gender, Height, and Weight data.
```

## `## Dataset` 

- `500 records` 

- `Features:` 

- `Gender` 

- `Height` 

- `Weight` 

- `BMI Category (Index)` 

```
BMI values were calculated during preprocessing using the standard BMI formula.
```

## `## Technologies Used` 

```
- Python
- Google Colab
```

```
- Pandas
```

- `NumPy` 

- `Matplotlib` 

- `Scikit-learn` 

## `## Model` 

```
Algorithm Used:
```

- `Linear Regression` 

## `## Results` 

```
| Metric | Value |
|---------|-------|
| MAE | 1.887 |
| RMSE | 2.567 |
| R² Score | 0.971 |
```

```
The model achieved high prediction accuracy, explaining approximately 97% of the
variance in BMI values.
```
